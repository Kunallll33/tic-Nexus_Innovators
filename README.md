# Homigo

Homigo is a simple full-stack home services marketplace built for hackathon demos.

## Tech stack

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js, Express
- Database: MongoDB, Mongoose

## Features

- Customer and provider registration/login
- City-based provider matching
- Booking system with status updates
- Customer and provider dashboards
- Simple admin overview page

## Run locally

1. Copy `.env.example` to `.env`
2. Add your MongoDB connection string
3. Install packages with `npm install`
4. Start the app with `npm run dev`
5. Open `http://localhost:5000`
